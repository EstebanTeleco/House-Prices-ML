# Datos

Esta carpeta va vacía en el repo (los CSV pesan y además son del dataset de Kaggle, no míos para redistribuir).

Para correr el proyecto:

1. Entrá a la competencia [House Prices - Advanced Regression Techniques](https://www.kaggle.com/c/house-prices-advanced-regression-techniques) en Kaggle.
2. Descargá `train.csv` y `test.csv`.
3. Ponelos acá adentro, en `data/`.

Con eso ya podés correr `python main.py` desde la raíz del proyecto.
